# String Algorithm Toolkit (Java)

A collection of **string manipulation algorithms implemented in Java**, designed to operate on **arbitrary user-provided input strings** through a simple command-line interface.

This project demonstrates how fundamental algorithmic operations can transform and analyze strings using structured program flow, input validation, and modular methods.

---

## Features
- Interactive **CLI menu system**
- Algorithms for **string manipulation**
- Input validation and error handling
- Modular algorithm execution
- Repeatable execution workflow

---

## Current Algorithms

### String Reverse
Reverses the order of characters in an input string.

**Example**

Input
'''
hello
'''

Output 
'''
olleh
'''
---

## How It Works

The program runs an interactive menu where users can:

1. Select an algorithm
2. Provide a string input
3. View the algorithm’s result
4. Re-run the algorithm or return to the main menu

The architecture separates:
- **User Interface logic**
- **Input handling**
- **Algorithm implementations**

This makes it easy to extend the system with additional string algorithms.
